学习路线

先会用，再讲深

01

看见代码

理解程序就是“按顺序执行的指令”，学会运行第一个 Python 文件。

02

装东西

用变量、列表和字典保存文字、数字、清单、账单这些真实数据。

03

做判断

让程序根据条件走不同路线，比如余额不足、分数等级、登录状态。

04

重复做

用循环处理很多条数据，让电脑做重复活。

05

变成作品

把知识拼成三个小项目，真正体验“我写了一个能用的东西”。

动画理解

程序像一条会发光的流水线

输入 变量 判断 循环 输出

记住一个核心画面

Python 做的事很朴素：接收输入，把数据存起来，按规则加工，最后输出结果。 学习时不要急着背概念，先问自己：“这段代码拿了什么？改了什么？最后给我什么？”
