# Python 零基础互动学习教程

入口文件：`index.html`

打开方式：

1. 双击 `index.html`，用浏览器打开。
2. 按课程顺序学习，每一章都有动画讲解、代码示例、练习和小测。
3. `examples/` 目录里放了可复制运行的 Python 示例。
4. `checklists/` 目录里放了学习打卡清单。

建议节奏：

- 每天 45-60 分钟。
- 先看动画，再手打一遍代码。
- 不要只复制，要故意改一改数字、文字和条件，观察结果。

学习路线：

1. 认识 Python 和开发环境
2. 变量、数据类型、输入输出
3. 条件判断
4. 循环
5. 列表、字典、字符串
6. 函数
7. 文件读写与异常
8. 模块和第三方库
9. 面向对象入门
10. 三个小项目：记账本、猜数字、待办清单
