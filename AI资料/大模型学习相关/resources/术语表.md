# 大模型应用术语表

## LLM

Large Language Model，大语言模型。用于理解和生成自然语言。

## Token

模型处理文本的基本单位。输入和输出都会消耗 token。

## Context Window

上下文窗口。模型一次回答时能看到的最大内容长度。

## Prompt

给模型的指令和上下文。

## System Prompt

系统级提示词，通常用于定义角色、边界和全局规则。

## Embedding

把文本转换成向量，用来做语义搜索、聚类、相似度计算。

## Vector Database

向量数据库，用于存储和检索 embedding。

## RAG

Retrieval-Augmented Generation，检索增强生成。先检索资料，再让模型基于资料回答。

## Rerank

重排。对初步检索结果重新排序，提高上下文质量。

## Fine-tuning

微调。用特定数据让模型更适应某类任务。

## SFT

Supervised Fine-Tuning，监督微调。

## LoRA

Low-Rank Adaptation，低秩适配。参数高效微调方法。

## QLoRA

在量化模型上做 LoRA，降低显存需求。

## Agent

能根据任务自主选择工具、执行多步骤流程的大模型系统。

## Tool Calling

工具调用。模型根据需要调用外部函数、API 或系统能力。

## Hallucination

幻觉。模型生成了看似合理但不真实或无依据的内容。

## Evaluation

评测。用固定问题集和指标衡量模型应用质量。
