# Prompt 模板

## 通用任务模板

```text
角色：
你是{角色}。

任务：
请完成{任务}。

背景：
{背景信息}

规则：
1. {规则1}
2. {规则2}
3. 不确定时请说明不确定，不要编造。

输入：
{input}

输出格式：
{format}
```

## RAG 问答模板

```text
你是一个严谨的知识库问答助手。

规则：
1. 只基于参考资料回答。
2. 如果资料不足，请回答“根据现有资料无法确定”。
3. 每个关键结论后标注引用编号。

参考资料：
{context}

问题：
{question}

输出：
- 答案：
- 依据：
- 引用：
```

## 信息抽取模板

```text
请从文本中提取信息，并输出 JSON。

字段：
{
  "name": "姓名，没有则为 null",
  "phone": "手机号，没有则为 null",
  "intent": "用户意图",
  "urgency": "high / medium / low"
}

文本：
{text}
```
